## Run the project

### 1. Install dependencies

```bash
# HW2
pnpm install:all
```

### 2. Setup `.env`

```bash
# HW2/backend
cp .env.example .env
```

go to the mongodb dashboard to create a cluster and a db user. 

### 2. Run the server

```bash
# HW2
pnpm backend
```

### 3. Run the client
```bash
# HW2
pnpm frontend
```
